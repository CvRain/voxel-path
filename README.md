# Voxel Path: Artisan's Realm

## 快速开始

### 运行测试场景

1. 在 Godot 编辑器中打开项目
2. 打开场景文件：`Scenes/DemoWorld/TinyWorld.tscn`
3. 点击运行场景 (F6) 或主运行按钮 (F5)

### 场景内容

TinyWorld 场景包含了基本方块的展示：

- **地面层**：16x16 的草方块地面 (y=0)
- **泥土方块**：位置 (2, 1-2, 2)，展示纯泥土纹理
- **草方块**：位置 (5, 1-2, 2)，展示顶部草地、底部泥土、侧面草皮纹理
- **橡木树**：位置 (8, 1-5, 2)，包含橡木原木树干和树叶
- **独立树叶**：位置 (11, 1-2, 2)，展示半透明树叶效果

### 方块尺度与集群

- **基础微方块**：世界只存在 0.25 × 0.25 × 0.25 米的微方块，地形、树木和后续雕刻都直接写入这一分辨率。
- **方块组（集群）**：为了便于世界生成与玩家交互，系统会把 4×4×4 个微方块（对应 1m³）视为一个方块组；噪声采样、放置和破坏默认都按方块组为单位进行。
- **可变集群大小**：玩家可在 1m³（默认）与 2m³（按住 `Shift`）之间切换一次性放置/破坏的方块组体积，`Player.CurrentPlacementClusterSize` 始终反映当前组的边长（以微方块计）。

### 操作控制

- **WASD**：移动
- **空格**：跳跃
- **Shift**：冲刺
- **鼠标**：视角控制
- **ESC**：释放/捕获鼠标
- **鼠标左键**：破坏准星所指向的方块组（默认 1m³）
- **鼠标右键**：放置一组石头方块（用于快速搭建）

屏幕中央常驻十字准星用于辅助瞄准，只有在鼠标捕获状态下显示。

### 已注册的方块类型

1. **Air** (空气) - 透明，不可见
2. **Dirt** (泥土) - 全面使用泥土纹理
3. **Grass** (草方块) - 顶部草地，底部泥土，侧面草皮
4. **OakLog** (橡木原木) - 顶底使用年轮纹理，侧面使用树皮纹理
5. **OakLeaves** (橡木树叶) - 半透明绿色树叶

### 技术特性

- ✅ 方块纹理图集系统
- ✅ 灰度图自动染色 (草方块、树叶)
- ✅ 多面纹理支持 (不同面使用不同纹理)
- ✅ 半透明方块支持
- ✅ 第一人称控制器
- ✅ 基本光照和阴影
- ✅ 1m 宏方块 + 0.25m 微方块混合建造体系
